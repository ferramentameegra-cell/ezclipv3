# Backgrounds Fixos

Este diretório contém as imagens de background que serão aplicadas automaticamente a todos os vídeos gerados pela plataforma.

## Arquivo Principal

- `ezclip-background.png` (ou `.jpg`) - Background padrão para todos os vídeos (9:16 - 1080x1920)

## Como Adicionar

1. Coloque a imagem de background neste diretório
2. Nomeie como `ezclip-background.png` (ou `.jpg`)
3. A imagem deve ter resolução 1080x1920 (9:16) para evitar distorção
4. O sistema aplicará automaticamente em todos os vídeos gerados

## Nota

A imagem será redimensionada automaticamente para 1080x1920 mantendo proporção (sem distorção).
